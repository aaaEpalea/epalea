# Live API Testing

Interactive API testing is available at [epalea.ai/docs/api/live-testing](https://epalea.ai/docs/api/live-testing).

The Redoc explorer shows all endpoints — including coming-soon endpoints with their stub responses —
so you can understand the full API contract before those endpoints go live.

## How to use

1. Visit `epalea.ai/docs/api/live-testing`
2. Enter your API key in the Authorize dialog
3. Select an endpoint and edit the request body
4. Click **Try it out** to run a live request

## Postman collection

Download the Postman collection from `epalea.ai/docs/api/postman` or
from `postman/epalea_api_v1.json` in the repository.
